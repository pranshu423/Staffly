## Quick Start
1.  **Install Dependencies:**
    ```bash
    cd server && npm install
    cd ../client && npm install
    ```
2.  **Seed Database (creates demo users):**
    ```bash
    cd server
    npm run seed
    ```
3.  **Start Development Servers:**
    - Server: `cd server && npm run dev`
    - Client: `cd client && npm run dev`
